Design an award-winning desktop web application called "FlowNode".

This is NOT a todo app.

This is NOT a mind map.

This is NOT a Notion clone.

This is a visual operating system for planning and reviewing your entire day.

The core concept:

Your day is a living graph.

Everything is connected through beautiful nodes.

Users literally build today's life visually.
